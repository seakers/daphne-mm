I. Scenario

Imagine you are a material scientist in an engineering firm. You specialize in designing mechanical metamaterials for high-strength light-weight components. Mechanical metamaterials are artificial structures with mechanical properties defined by their structure.

To help you with your job, your firm recently acquired a decision support tool. The purpose of using this design support tool is to help you learn about designing mechanical metamaterials and create feasible metamaterials with high stiffness and low volume.

Accordingly, you have two goals in this experiment.

  • The first goal is to learn as much as possible about the problem at hand. There will be a test with questions after using the tool. Maximize your score by answering as many questions correctly as possible.
  • The second goal is to maximize the vertical stiffness and minimize volume fraction of mechanical materials as much as possible while making sure metamaterials are feasible.


II. Mechanical Metamaterial Design Problem

The tool focuses on a specific type of mechanical metamaterial which is defined using its unit cell.

  • The unit cell structure is made of multiple links joining nodes in a 3 x 3 grid. Not all links are necessary.
  • A unit cell structure repeats in a 2D plane to create a metamaterial, as seen from Figure below.
  • Every metamaterial is rated using two objectives, vertical stiffness (relates to strength) and volume fraction (relates to weight).
    Additionally, there is a feasibility constraint that the metamaterial should satisfy. Desired metamaterials are feasible and maximize stiffness and minimize volume fraction. All metrics are displayed as adimensional quantities normalized between 0 and 1.
    • Vertical Stiffness Relates to the amount of vertical force required to expand or contract a metamaterial by unit distance vertically The higher the better
    Volume Fraction The volume of unit cell structure divided by total volume (including missing links and empty space in the unit area) The smaller the better
    Feasibility (Constraint) No two links in a unit cell should intersect, except at nodes. Also, the resulting metamaterial should be fully connected (i.e. it cannot have disconnected subcomponents). Full feasibility (i.e., no intersections or disconnected components) is desired


III. Tool

Using this tool, you can explore the dataset of existing metamaterials and of metamaterials you create. To view a metamaterial, click on corresponding point on the tradespace plot. Upon clicking, the selected point is marked by big red plus sign (➕) and the corresponding metamaterial along with relevant information appears in the bottom left window.

The blue dotted line represents a set of optimal designs, also called the Pareto front . A design belongs to the Pareto front if there exists no other design in the set that has better stiffness for the same or lower volume fraction and vice versa. For example, the metamaterial design shown here with the red plus symbol is dominated by the Pareto designs on the blue dotted line.

Only feasible metamaterials are considered part of the Pareto front.

To modify a selected metamaterial, the tool offers three functionalities: (i) Change Design, (ii) Change Features, and (iii) Automated Feature Changes

The tool uses a total of 10 predefined features for metamaterials. Five randomly selected features are used in the "Change Feature" functionality and the rest are used in the "Automated Feature Changes" functionality. Some features are abstract features created using a machine learning algorithm. All features take value between -2.5 (low) to 2.5 (high).

  1. Change Design
    You can change the selected metamaterial or create your own metamaterial using "Change Design" feature. Press "Test Metamaterial" to evaluate the design objectives at the new metamaterial.
  2. Change Features
    Once you select a metamaterial from the tradespace plot, the features of the selected metamaterial appear in the "Change Features" window. You can change values of any feature you like (patterns contained by many good designs) and the tool will generate a new metamaterial with new feature values.
  3. Automated Feature Changes
    Finally, the tool offers a functionality to change given features automatically. To use this functionality, you will need to select the amount of "overall change" you desire in features of the selected design. For the selected design and overall change, the tool will automatically suggest good changes in the feature values and generate a new metamaterial.
